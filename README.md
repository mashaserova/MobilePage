# Layout Test Assignment

Реализация верстки мобильной страницы согласно техническому заданию с использованием React, TypeScript и Material UI.

## Демонстрация
Проект развернут по адресу: https://mobile-page-675e.vercel.app/

## Стек технологий
- React
- TypeScript
- Material UI (MUI) v5

## Детали реализации
- **Адаптивность:** Верстка подстраивается под разрешение экрана в диапазоне 320px - 600px.
- **Темизация:** Основные стили (цвета, шрифты, скругления) вынесены в глобальную тему MUI для минимизации использования sx-пропсов.
- **Компоненты:** Интерфейс разбит на переиспользуемые компоненты.
- **Логика:** Реализована только визуальная часть, взаимодействие с формами не предусмотрено ТЗ.

## Запуск проекта

### Требования
*   Node.js (версия 18.x или выше)
*   npm / yarn

### Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ВАШ_НИКНЕЙМ/ВАШ_РЕПОЗИТОРИЙ.git
    cd ВАШ_РЕПОЗИТОРИЙ
    ```

2.  **Установите зависимости:**
    ```bash
    npm install
    ```

3.  **Запустите приложение в режиме разработки:**
    ```bash
    npm run dev
    ```
    Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере.
